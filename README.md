# CREATING A LIST

Unorderd list: you can create an unorderd list item by preceeding the list item with either a -, + or *.

+ Number 1
   + Nested 1
   + Nested 2
+ Number 2
+ Number 3

- Number 1
   - Nested 1
   - Nested 2
- Number 2
- Number 3

* Number 1
* Number 2
* Number 3

Ordederd LIst: Orderd list can be created using numbers.

1. Item one
2. Item two
3. Item three
4. Item four
   1. Item four nested 1
   2. Item four nested 2
